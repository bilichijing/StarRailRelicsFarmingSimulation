# 遗器刷取副词条达标模拟器

这是一个用于模拟《崩坏：星穹铁道》遗器刷取过程的Python程序，可以计算获得指定副词条数量的概率分布，并将结果导出到Excel文件。

## 功能特点

- 模拟遗器刷取过程，包括外圈和内圈遗器
- 支持自定义筛选条件（套装、部位、主属性）
- 支持自定义有效副词条类型
- 自动强化遗器到+5（可选）
- 生成详细的概率分布统计
- 将结果导出到Excel文件，包含多个工作表

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 配置 `test_config.json` 文件：
   - 设置刷取参数（副本、次数、是否强化等）
   - 设置筛选条件（需要的遗器套装、部位、主属性）
   - 设置有效副词条类型

2. 运行程序：
   ```bash
   python 遗器刷取模拟.py
   ```

3. 查看结果：
   - 程序会在控制台输出概率分布
   - 同时生成Excel文件 `遗器刷取统计表.xlsx`

## Excel输出文件说明

生成的Excel文件包含以下工作表：

1. **概率分布** - 各有效副词条数的出现次数和概率
2. **配置信息** - 本次模拟的配置参数
3. **统计摘要** - 达到不同副词条数的累积概率
4. **详细统计** - 详细的统计数据

## 配置示例

```json
{
  "刷取设置": {
    "外圈副本次数": 120,
    "外圈副本": "外圈副本1",
    "内圈副本次数": 80,
    "内圈副本": "内圈副本2",
    "是否强化": true,
    "模拟次数": 1000
  },
  "筛选条件": [
    {
      "套装": "风套",
      "部位": "头",
      "主属性": "生命"
    }
  ],
  "有效副词条": [
    "暴击",
    "暴伤",
    "攻击百分比"
  ]
}
```

## 注意事项

- 模拟次数建议设置为1000次以上以获得更准确的统计结果
- 程序会自动处理遗器的副词条抽取和强化过程
- Excel文件会覆盖同名文件，请注意备份
